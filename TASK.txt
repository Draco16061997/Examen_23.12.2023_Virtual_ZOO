Задание: Разработка системы управления виртуальным зоопарком

Задача: Создать консольное приложение, которое моделирует зоопарк с различными животными, их поведением и взаимодействиями внутри зоопарка.

Базовый класс Животное (Animal):
Атрибуты: вид, имя, возраст, состояние здоровья, уровень голода и т.д.
Методы: конструкторы, методы для имитации поведения (например, кормление, движение, сон)

Производные классы для различных видов животных (например, Лев, Слон, Пингвин):
Уникальные атрибуты и методы, характерные для каждого вида.
Реализация специфических поведенческих моделей.

Класс Вольер (Enclosure):
Атрибуты: размер, тип местности, список животных внутри.
Методы: методы для управления животными внутри вольера, учет условий среды.

Класс Зоопарк (Zoo):
Атрибуты: список вольеров, сотрудники, финансовые данные.
Методы: управление вольерами, учет посетителей, финансовое управление.
Система событий и взаимодействий:

Реализация системы событий, таких как кормление, уход за животными, реакции на погодные условия.
Взаимодействие между животными (например, социальное поведение, конкуренция за ресурсы).
Интерфейс управления зоопарком:

Возможности для пользователя управлять зоопарком, включая принятие решений о покупке новых животных, строительстве вольеров и управлении ресурсами.
Дополнительные функции:

Расширенные алгоритмы для моделирования экологических и биологических процессов в зоопарке.
